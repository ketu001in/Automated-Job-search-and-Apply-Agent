🚀 Getting Started

What Nexora is and how a typical session flows

Nexora automates applying to LinkedIn jobs through the Easy Apply button, using the profile and preferences you save in the tool. It runs entirely on your own computer — there is no server, no subscription, and no data sent anywhere except to LinkedIn itself.

Typical flow

  • Login screen — pick your profile card, or create a new one.
  • First time only: a short setup wizard collects your details.
  • Dashboard — click Run Now whenever you want Nexora to search and apply.
  • My Profile — update your details any time after setup.

Multiple people can use Nexora on the same Windows PC — each person's data, LinkedIn login session, and application history stay completely separate.

👤 Logging In & Profiles

The screen you see every time Nexora starts

What you'll see

  • A card for every person who has used Nexora on this computer, showing their name and a masked email.
  • Click a card to log in instantly — your saved profile, preferences, and report history load right away.
  • Create new profile — enter a name and LinkedIn email to set up a brand new, separate profile.

Switching between profiles

  • ⇄ Switch (on the Dashboard) — returns to this login screen so a different person can log in.
  • ✓ Sign Off — also returns to the login screen. Use this when you're done for the day; your data is saved automatically either way.

Every LinkedIn account also gets its own private Chrome session — switching users never reuses someone else's LinkedIn login, and you never need to log out of LinkedIn manually.

🧭 First-Time Setup

Shown exactly once, the first time a new profile is created

The setup wizard walks through 10 short steps: LinkedIn credentials, job search status, IT / Non-IT category, CV upload, personal details, address, education, experience, job preferences, and schedule.

About the CV upload step

Uploading your CV auto-fills most of the following steps for you. Always review every step before finishing — automatic extraction is a head start, not a guarantee of accuracy.

After you finish

Once you click Save & Finish, setup is complete for that profile — you will never see the wizard again. All future changes are made in My Profile, which is edit-only by design.

📝 My Profile

The only place you edit your data after setup

Sections

Use the left panel to jump between: IT/Non-IT Category, LinkedIn, CV/Resume, Personal, Address, Education, Experience, Job Preferences, and Travel & Schedule. The section you're currently viewing is highlighted as you scroll.

Critical: values come only from what you enter here

Fields like Notice Period, Current CTC, and Expected CTC are used exactly as you enter them — Nexora never substitutes a default or made-up value. If a field is left blank, Nexora leaves the matching LinkedIn question blank too rather than guessing, so it's worth keeping these filled in before running.

Changing your LinkedIn email here

If you change your LinkedIn email in Settings (credentials live there), Nexora checks whether it recognizes that email — see the Settings section below.

⚙️ Settings

LinkedIn credentials and run behavior

LinkedIn credentials

If you change the email to one Nexora has seen before, it offers to switch you to that saved profile. If it's a brand-new email, it offers to create a fresh profile — your current profile is left untouched either way.

Run behavior

  • Show Chrome window — recommended for your first few runs so you can watch fields fill in.
  • Max applications per run — a safety cap so a single run doesn't apply to an unbounded number of jobs.
  • Minimum compatibility score — jobs scoring below this are skipped, not applied to.
  • Job search time window — how recently a job must have been posted (24 hours / 3 days / week / month).

📊 Dashboard

Where you start runs and review results

Running the agent

Run Now starts a search-and-apply cycle using your current My Profile data. All stats shown are per-user — you only ever see your own counts, never another profile's.

Reading the results

  • All Applied Jobs — click the "Total Applied" number to jump straight to the list; export as CSV, XLS, or PDF.
  • Skipped Jobs — jobs that scored below your minimum compatibility threshold.
  • 📌 Found These Jobs — Apply Manually — jobs that matched your profile but don't have LinkedIn's Easy Apply button. Click the title to open and apply directly on LinkedIn.
  • 🧠 Self-learning active banner — shows how many form questions Nexora has learned answers to over time, across all your runs.

⏸ Pause, Resume & Floating Control

Staying in control while a run is in progress

The floating control bar

While a run is active, a small window appears in the bottom-right of your screen, always on top of Chrome — so you never have to switch windows to manage the run.

  • ⏸ Pause / hotkey Ctrl+Alt+P — pauses after the current job finishes (never mid-fill). Correct any field directly in Chrome while paused.
  • ▶ Resume / same hotkey — continues exactly where it left off.
  • 📊 Dashboard ↔ 🌐 Return to Chrome — this button swaps automatically depending on which window currently has focus, always offering the other one.
  • ⏹ Force Stop — only for when the agent appears frozen. Safely closes the browser and resets everything to Idle.

✅ Do's & Don'ts

Quick rules of thumb for reliable runs
DO
  • Keep My Profile fully filled in — especially Notice Period and CTC — before running.
  • Watch the first few runs in the visible Chrome window.
  • Use Pause the moment you spot a wrong field, rather than letting it continue.
  • Keep your LinkedIn password current in Settings.
DON'T
  • Don't run two profiles applying as the same LinkedIn account at once.
  • Don't force-close Chrome if you can help it — use Pause or Force Stop instead.
  • Don't expect Nexora to apply where there's no Easy Apply button — those go to the manual list.
  • Don't leave CTC/Notice Period blank and expect a sensible default — Nexora leaves those questions blank too.

❓ Frequently Asked Questions

Common questions, answered plainly
Will Nexora apply to jobs I'm not qualified for?
No — every job is scored against your profile and skipped if it's below your Minimum Compatibility Score (set in Settings).
What happens to LinkedIn questions that aren't in my profile?
Nexora checks its self-learning question bank (built from your own past answers) first, then leaves the field blank rather than guessing.
Can two people use Nexora on the same computer?
Yes — each person gets their own login card, and their profile, LinkedIn session, and applied-jobs history stay completely separate.
I changed my LinkedIn password — do I need to update anything?
Yes, update it in Settings or My Profile, otherwise the next run will fail to log in.
Where is my data stored?
Entirely on your own computer, in your Windows user profile folder — never on a server.
The agent looks stuck — what do I do?
Use ⏹ Force Stop on the floating control bar or Dashboard. It safely closes the browser and resets to Idle.

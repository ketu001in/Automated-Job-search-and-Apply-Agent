WHAT IS NEXORA?
━━━━━━━━━━━━━━━
Nexora is an intelligent LinkedIn job application agent that automates
your job search. Upload your CV once, set your preferences, and let
Nexora apply to matching jobs — automatically, accurately, and smarter
with every run.

  ✅ Parses your CV and auto-fills all profile fields
  ✅ Applies to matching LinkedIn Easy Apply jobs automatically
  ✅ Self-learning: builds a Q&A knowledge base from every application
  ✅ IT / Non-IT profile categorisation for targeted job matching
  ✅ Date-grouped dashboard with CSV / XLS / PDF export
  ✅ Multiple people can share one PC — every profile, LinkedIn session,
     and application history stays completely private and separate
  ✅ Pause/resume any run without leaving Chrome (Ctrl+Alt+P)
  ✅ 100% local — your data never leaves your device
  ✅ Zero cost — no API keys, no subscriptions

WHAT DOWNLOADS ON FIRST LAUNCH

INSTALLATION
━━━━━━━━━━━━
  1. Double-click  "Nexora Setup 1.0.0.exe"
  2. Follow the installer wizard
     • Choose installation folder (default: C:\Program Files\Nexora\)
     • Choose whether to create a Desktop shortcut
  3. Click "Finish" — Nexora launches automatically

  Windows SmartScreen Note:
  ─────────────────────────
  On first run you may see "Windows protected your PC."
  Click "More info" → "Run anyway".
  This is normal for unsigned installers. Nexora is safe.
  (Code signing certificate will be added in v2.0.)

FIRST-TIME SETUP (Setup Wizard)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Step 1 : Welcome screen
  Step 2 : Enter your LinkedIn email and password
  Step 3 : Select your job search status
  Step 4 : Upload your CV (PDF or DOCX, max 5 MB)
            → Nexora auto-fills all fields from your CV
  Step 5 : Review / edit Personal Details
  Step 6 : Review / edit Address
  Step 7 : Review / edit Education history
  Step 8 : Review / edit Work Experience
  Step 9 : Set Job Preferences (target roles, locations, CTC)
  Step 10: Set Travel preference & Automation Schedule

  Click "Save & Launch Dashboard" — you're ready!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

RUNNING THE AGENT
━━━━━━━━━━━━━━━━━
  Manual:    Click "▶ Run Now" on the Dashboard
  Scheduled: Set a schedule in Settings → fires automatically
  Tray:      Right-click the system tray icon → "Run Now"

  On first run, Nexora will:
  1. Download Chromium (~170 MB) — takes 1-3 minutes
  2. Open a browser window (if "Show browser" is enabled)
  3. Log in to LinkedIn with your saved credentials
  4. Handle mobile 2FA verification (shows prompt in Live Log)
  5. Search and apply to matching jobs
  6. Show results in the Dashboard

SELF-LEARNING FEATURE
━━━━━━━━━━━━━━━━━━━━━
  Nexora remembers every question it answers on a job application form.
  After each successful application, the Q&A is saved to a local
  knowledge base. On subsequent runs, Nexora recalls these answers —
  getting more accurate with every application.

  The knowledge base grows automatically. No action needed.

IT / NON-IT PROFILE CATEGORY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TROUBLESHOOTING
━━━━━━━━━━━━━━━
  Problem:  "Login form did not appear"
  Fix:      In Settings → enable "Show browser while applying" →
            Run Now → log in manually in the browser window.
            Session is saved for future runs.

  Problem:  LinkedIn asks for mobile verification every run
  Fix:      Normal on first run with a new browser. After approving
            once on your phone, the session cookie is saved and
            future runs will not ask again.

  Problem:  Chromium not downloading
  Fix:      Ensure internet is available. Run from an Admin terminal:
            npm run setup-browser

  Problem:  "No Easy Apply button" for most jobs
  Fix:      LinkedIn may have changed their UI. Update target roles
            in My Profile. Some jobs only show External Apply.

DATA PRIVACY
━━━━━━━━━━━━
  All your data is stored locally on your Windows device:
    Config:   C:\Users\<Name>\AppData\Roaming\nexora\
    Browser:  C:\Users\<Name>\AppData\Roaming\nexora\browser-profile\
    Chromium: C:\Users\<Name>\AppData\Local\ms-playwright\

  Nexora does NOT send any data to external servers.
  Bosket's Tech Ventures has zero access to your profile or CV.
